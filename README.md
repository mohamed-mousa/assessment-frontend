## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/mohamed-mousa/assessment-frontend
   ```

2. Navigate to the project directory:

   ```sh
   cd assessment-frontend
   ```

3. Install the dependencies:

   Using npm:

   ```sh
   npm install
   ```

   Using Yarn:

   ```sh
   yarn install
   ```

## Running the App

To start the development server:

Using npm:

```sh
npm run dev
```
